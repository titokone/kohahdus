\subsection{\label{fi.helsinki.cs.kohahdus.DBHandler}\index{DBHandler}Class DBHandler}{
\vskip .1in 
Singleton class used for database interactions. Each public method of DBHandler class encapsulates one database transaction, and thus may cause multiple inserts/updates/removes with one call. The atomicity of the operations is quaranteed by using the transaction model provided by the SQL standard.\vskip .1in
